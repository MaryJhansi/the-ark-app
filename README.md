# The Ark 🖊️

A responsive one-page website for **The Ark** — a stationery, 
gifts, toys, books and craft shop.

## 🛠 Built With
- HTML5
- CSS3 (Flexbox, CSS Variables, Media Queries)
- JavaScript (DOM Manipulation)
- Google Fonts (Playfair Display + Lato)

## ✅ Completed Sections
- [x] Navbar (responsive with hamburger menu)
- [x] Hero Section (gradient, animated GIF, CTA button)
- [x] About Section (brand story, value cards)

## 🚧 In Progress

- [ ] Featured Products
- [ ] Contact Section (with WhatsApp)
- [ ] Footer

frontend development skills and build a professional portfolio piece.
